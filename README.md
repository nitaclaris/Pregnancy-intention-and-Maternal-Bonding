# Maternal Attitudes and Infant Bonding in AGYW

## Project Overview

This repository contains the Stata scripts used to investigate the relationship between pregnancy intendedness, maternal attitudes, maternal adjustment, and mother–infant bonding among adolescent girls and young women (AGYW) in southwestern Kenya.

The analysis was conducted using data from an ongoing  clinical trial, and evaluates how unintended pregnancy influences psychosocial outcomes during pregnancy and the postpartum period.

## Objectives

- Assess the association between pregnancy intendedness and maternal adjustment.
- Examine maternal attitudes toward pregnancy, the baby, and sexual relationships.
- Determine whether unintended pregnancy is associated with impaired mother–infant bonding.
- Identify demographic and psychosocial predictors of adverse maternal outcomes.

## Methods

- Comparative cross-sectional study
- Nested within an ongoing Clinical trial
- Sample size: Parenting AGYW
- Multiple linear regression
- Logistic regression

## Key Findings

- Unintended pregnancy was associated with less positive maternal attitudes toward sex after adjustment for demographic and psychosocial factors.
- Parenting AGYW with unintended pregnancies had more than five times higher odds of impaired mother–infant bonding than those with intended pregnancies.
- Older maternal age and a higher number of previous pregnancies were independently associated with poorer mother–infant bonding.
- Forced marriage, lower socioeconomic status, and previous experiences of loss were associated with poorer maternal adjustment and attitudes.
- The findings emphasize the importance of psychosocial support for young mothers experiencing unintended pregnancies.

## Data

The dataset used in this project is not publicly available because it originates from an ongoing clinical trial. In accordance with the study's data sharing policies, participant-level data cannot be publicly released.

This repository contains only the Stata analysis scripts and project documentation.
